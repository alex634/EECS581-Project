ShipGen

Author: Harry W Date: Sep 27, 2024 Last Modifited: Sep 27, 2024 Purpose: this program assumes a 10x10 grid which we can use an index from 0 to 99 inclusive from there based on the index the row and col can be calculated and checked for ship collisions

ShipGen.gen_ship(num_ship)

The function generates the ship locations and rotation based on the numbers of ships passed in and returns a tuple of the head of the ship and if the ship are rotated.

ShipGen.int_to_coord(value)

The function converts an interger value to a string for the column int for row

ShipGen.place_ships(placed_ships, value, ship_size, rotate)

The function add a ship to a list and marks that location as taken so that there won’t be two ship on top of each other.
